📸 Valentine's Day Photo Booth 💖
This project was created especially for Valentine's Day! It is a simple photo booth built using HTML and CSS to celebrate love and special moments.

🎭 About the Project
A fun photo booth effect with animations.
Fully designed using HTML & CSS.
Easy to customize with your own images.

🖼️ Customizing Your Photos
To personalize the experience, I’ve included a photoEdit file, where you can replace the default images with your own. Just swap the files, and your personalized photo booth is ready!
